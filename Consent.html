Permiso para visitar y Consentimiento para evaluar

Se debe obtener permiso y consentimiento antes de asignar al paciente. Un Despachador no puede asignar a un paciente sin consentimiento o permiso. La sección de Consentimiento de la Herramienta de incorporación se enfoca en garantizar que capturemos y documentemos la siguiente información:

  • El tomador de decisiones médicas para el paciente.
  • Si el tomador de decisiones médicas del paciente estará en la escena.
  • Si tenemos consentimiento verbal para visitar, tratar y comunicarnos con el paciente.

Obtención de permiso y consentimiento

  1. Establecer consentimiento verbal

    Debemos asegurarnos de tener el consentimiento verbal para visitar al paciente. Al establecer el consentimiento, significa que tenemos permiso para visitar el pacient a su dirección.

  2. Confirmar la comunicación con el paciente

    Debemos confirmar que podemos comunicarnos con el paciente a través de correos de voz, mensajes de texto, etc. La captura de este consentimiento confirmará si este solicitante recibe comunicación continua de DispatchHealth, incluida la aplicación Companion.

  3. Confirme quién toma las decisiones médicas para este paciente. Establezca si se debe realizar una llamada por separado al tomador de decisiones médicas antes de la visita.

    El despachador hará un seguimiento con las preguntas que se le soliciten en la herramienta de incorporación para confirmar esta información.
    Nota: Si el paciente dice "no" a ciertas preguntas, presentaremos el formulario de Resolver Razón para que el caso sea debidamente dispuesto.

  4. Si el tomador de decisiones médicas está disponible durante la llamada, pida hablar con él para continuar. Cuando el paciente/tomador de decisiones médicas se ponga en línea, preséntese y hágales saber que la llamada está grabada.

    "Hola (paciente/tomador de decisiones médicas), soy (inserte su nombre) de DispatchHealth y me gustaría que supiera que esta llamada se graba con fines de control de calidad. Si podemos programar esta cita, ¿tenemos su consentimiento? tratar (usted/Paciente)?

  5. Si el tomador de decisiones médicas no está al teléfono y no estará en el lugar, la herramienta le indicará que llame al tomador de decisiones médicas. Capture el consentimiento verbal para seguir adelante con la asignación.

    "Hola (tomador de decisiones médicas), soy (inserte su nombre) de DispatchHealth y me gustaría que sepa que esta llamada se graba con fines de control de calidad. Si podemos programar esta cita, ¿tenemos su consentimiento para tratar (usted/Paciente)?"
